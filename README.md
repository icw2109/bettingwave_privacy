# BettingWave Policy Site

This repository hosts public policy documents for the BettingWave app.

- Public URL (after enabling GitHub Pages):
  - https://<your-username>.github.io/bettingwave-policy/
  - Policy page: https://<your-username>.github.io/bettingwave-policy/privacy-policy.html

## Structure
- `docs/` — static website root for GitHub Pages
  - `privacy-policy.html` — the Privacy Policy page
  - `index.html` — redirect to `privacy-policy.html`
- `.nojekyll` — disables Jekyll processing on GitHub Pages

## Publishing (GitHub Pages)
1. Push to a new GitHub repository (e.g., `bettingwave-policy`).
2. In GitHub → Settings → Pages:
   - Source: Deploy from a branch
   - Branch: `main` (or `master`)
   - Folder: `/docs`
3. Wait ~1–3 minutes for the site to go live.
